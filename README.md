# VisionNavi

VisionNavi is an AI-powered assistive navigation application for visually impaired individuals.

Using a smartphone or laptop camera, VisionNavi continuously captures images of the user's surroundings and sends them to Anthropic Claude Vision AI for analysis. The system identifies obstacles, hazards, and important navigation information, then provides spoken feedback through text-to-speech.

Key features include:

* Real-time scene analysis
* Obstacle and hazard detection
* AI-generated navigation descriptions
* Distance estimation for nearby objects
* Audio guidance and warnings
* Voice command controls
* Proximity radar visualization

VisionNavi aims to increase independence and safety for visually impaired users by transforming visual information into actionable audio guidance.
